# 💧 Health Pro - Fitness & Hydration Tracker

A modern, gamified mobile application built with Flutter to help users monitor their daily health targets. It features a polished Material 3 UI, dynamic analytics, and local data persistence.

## ✨ Key Features
* **Gamification:** Rewards users with a 🎉 Confetti animation and tracks their daily streak upon reaching their hydration or activity goals.
* **Data Persistence:** Utilizes `SharedPreferences` to save user goals, logs, and streak counts locally, ensuring data remains intact across app sessions.
* **Data Visualization:** Integrates `fl_chart` to render dynamic, interactive bar charts for analyzing recent activity logs.
* **Modern Theming:** Fully responsive **Dark & Light modes** with implicitly animated gradient progress rings and glass-like card elevations.
* **Clean Architecture:** Built using the `Provider` package to cleanly separate the business logic from the UI state.

## 🛠️ Tech Stack
* **Framework:** Flutter / Dart

* **Storage:** SharedPreferences
* **Visuals:** Confetti, Fl_Chart

## 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/your-username/health-pro-fitness-tracker.git`
2. Install dependencies: `flutter pub get`
3. Run the app: `flutter run`
